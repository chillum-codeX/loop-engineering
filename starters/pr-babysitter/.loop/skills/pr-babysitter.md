# pr-babysitter

## When

Run on demand or whenever an open pull request changes.

## Read

- Open pull requests
- Review decisions
- Check-run status
- Mergeability and last-update time

## Judge

Identify drafts, merge conflicts, missing approvals, requested changes,
failing checks, pending checks, and stale pull requests.

## Output

Write a deterministic pull-request health report for human review.

## Stop

- Do not merge pull requests
- Do not dismiss reviews
- Do not modify branches
- Do not request reviewers without explicit approval
