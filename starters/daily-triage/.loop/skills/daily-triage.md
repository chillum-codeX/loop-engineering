# daily-triage

## When

Daily at 9 AM or when manually triggered.

## Read

- Open GitHub issues
- Open pull requests
- Slack messages with 🔥 emoji
- Today's calendar events

## Judge

Success criteria:
- All issues have priority labels (P0, P1, P2)
- All PRs have assigned reviewers
- Calendar conflicts are flagged
- Blockers are identified and tagged

## Output

- Labeled issues
- Assigned PRs
- Slack summary to #daily-updates
- Updated project board

## Stop

- Do NOT close issues without human approval
- Do NOT merge pull requests
- Do NOT exceed 10,000 tokens
- Stop after 5 iterations

## Tool Configuration

Optimized for: Claude Sonnet
